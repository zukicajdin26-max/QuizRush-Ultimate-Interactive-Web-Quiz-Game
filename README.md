# 🎮 QuizRush Ultimate

QuizRush Ultimate is a modern, interactive browser-based quiz game designed with clean UI, engaging gameplay, and smooth user experience. It is built using HTML, CSS, and JavaScript and is fully responsive across devices.

## 🎯 Features

* 🧠 Multiple quiz categories (General, Math)
* 📊 Score & level system
* ⏱ Countdown timer
* 🔊 Sound effects for correct/wrong answers
* 🎨 Smooth animations and modern UI
* 📱 Fully responsive (mobile-friendly)
* 🎮 Game over screen with restart system

---

## 🎮 Gameplay

* Choose a category
* Answer questions before time runs out
* Gain points and level up
* Wrong answer or timeout = Game Over
